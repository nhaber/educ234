Deep Learning

The Tensorflow tutorials are extensive, well-organized, and integrated with Colab (you can run everything on cloud services, for free, without thinking about it!) --- I'd highly recommend. It has some reinforcement learning bits as well. PyTorch tutorials are also great.

For an essentially code-free demo, check out Teachable Machine. Papers with Code provides a massive compilation of benchmarks.

Reinforcement Learning

Stable baselines provides a wide range of reinforcement learning implementations. The Neptune AI blog has a list of benchmarks (but this is very inexhaustive...).
